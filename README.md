## Installation

```bash
# Clone the repository
git clone YOUR_REPO_URL
cd doctors-app

# Install dependencies
npm install

# Start the server
node server.js
```

## Usage

Open your browser and navigate to:
- Reception: `http://localhost:3000/reception.html`
- Doctor: `http://localhost:3000/doctor.html`
- Pharmacy: `http://localhost:3000/pharmacy.html`

## Tech Stack

- **Backend:** Node.js, Express
- **Database:** SQLite3
- **Real-time:** Socket.IO
- **Export:** ExcelJS
- **Frontend:** HTML, CSS, JavaScript

## Deployment

This app requires a Node.js server. Deploy on:
- Render.com (Recommended)
- Railway.app
- Heroku
- Your own VPS
